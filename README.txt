
Mostudio-like Dark Portfolio (Arabic/English)
============================================
المحتويات:
- index.html  : الصفحة الرئيسية
- assets/css/style.css : تنسيقات مخصصة (Dark)
- assets/js/main.js : تبديل اللغة + فلترة الفئات + معاينة الصور
- assets/images/ : ضع صورك هنا (إن أردت استخدام صور محلية)

طريقة التشغيل:
1) افتح المجلد في VS Code.
2) ثبّت إضافة Live Server (اختياري لكنه مريح).
3) انقر يمين على index.html ثم Open with Live Server.
4) لتغيير البيانات:
   - الهاتف/الإيميل/العنوان: داخل قسم #contact.
   - روابط السوشيال: أعلى الشريط الجانبي.
   - الصور: عدّل روابط الصور في قسم #portfolio أو ضع صورك داخل assets/images واكتب المسارات.
5) اللغات:
   - زر "English/العربية" يبدّل بين العربي والإنجليزي ويغير اتجاه الصفحة RTL/LTR.
6) الفئات (مطاعم/شركات/مناسبات):
   - استخدم أزرار الفلترة أعلى قسم الأعمال.
